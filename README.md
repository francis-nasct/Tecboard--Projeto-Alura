<p>A Landing Page do projeto fictício Tecboard foi desenvolvida durante o curso de HTML e CSS da Alura, focado em Ambiente, Estrutura e Estilo.

Com este projeto, compreendi os fundamentos do HTML e a semântica das tags, apliquei estilos e técnicas de responsividade com CSS e aprendi a organizar e estruturar o código para facilitar a manutenção. 

Além disso, utilizei o Git e GitHub para controle de versões e gerenciar mudanças no código, através de comandos essenciais do terminal para navegar e manipular diretórios. 

Por fim, hospedei o site estático na plataforma da Vercel.
</p>
